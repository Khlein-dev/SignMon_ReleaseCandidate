# Mini-Pay App

For hackathon

## Ideas

### AI Financial Coach (Chat-based)

    Problem solved: lack of financial literacy

    Idea:
    A chatbot that acts like a financial mentor.

    Features:

    Ask: “Can I buy this?”
    Gives advice based on income & savings
    Explains concepts simply (like you're a beginner)
    Weekly financial health check


### Gamified Savings App (Like a Game)

    Problem solved: no motivation to save

    Idea:
    Turn saving money into a game with levels, rewards, and streaks.

    Features:

    Daily/weekly saving challenges
    XP system (save ₱100 = gain points)
    Unlock badges or avatars
    “Don’t break the streak” mechanic

    Extra twist:
    Add a friend leaderboard to create healthy competition.
